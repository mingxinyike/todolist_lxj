#demo1 
- 使用了 NEJ 的 DOM 操作、事件监听和 JST 模板
- MCSS 解析成 css
- 通过 localStorange 存储数据

#demo2
- 使用 RegularJS MVVM 框架，完成 data 到 view 的自动更新

#demo3
- 在demo1的基础上将数据存储到 mongodb 数据库中
- 通过 NEJ 的 util.ajax 模块发送数据请求
- nodejs 链接 mongodb 数据库， 完成增、查、删、改（单个、批量）操作。
